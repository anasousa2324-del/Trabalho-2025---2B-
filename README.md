# Trabalho-2025---2B-

Este trabalho consiste na continuidade do desenvolvimento de um sistema web integrado a um banco de dados, iniciado em laboratório. O objetivo é criar um sistema funcional, com interface em PHP, capaz de inserir, consultar e apresentar informações armazenadas no banco.

O projeto apresenta as seguintes etapas:
1. Criar um sistema que alimente o banco inserindo as informações através de uma
interface web com PHP, pode-se utilizar o PHPMyAdmin para a conexão e definir
pelo menos 10 consultas para o banco, seu sistema também deve ter gráficos
gerados a partir dos dados que estão no banco, com informações que sejam úteis.
2. Essas consultas devem ser apresentadas para o usuário de algumas formas,
podendo ser com graficos, cards, relatórios ou a combinação dessas
possibilidades.
3. Você deve popular o banco com pelo menos 100 linhas/tuplas, procure uma forma
de alimentar esse banco sem necessariamente inserir via formulário.
4. Todas essas informações devem estar em um repositório do github com imagens e
códigos como uma introdução no readme do repositório, onde o estudante colocará
os códigos do sistema, SQL, consultas criadas e demais detalhes.
